#Project Title:- 
Admin Panel in Sequelize using Mysql Database


#How to Run Project:-

1) Install npm i :- for download all packages
2) This is PORT=3000
3) Import Postman collection
4) Import sql file in phpmyadmin
5) After that, Run all APIs 

#Name of Packages:-

1)@hapi/joi
	For Field Validation
2)bcrypt
	For Password Security
3)body-parser
	 This is a node.js middleware for handling JSON, Raw, Text and URL encoded form data.
4)dotenv
	In .env add secure data
5)express
	Express.js is a web application framework for Node.js. It provides various features that make web application development fast and easy which otherwise takes more time using only Node.js.
6)express-joi-validation
	Its using for validating Express request object.
7)jsonwebtoken
	For Generate Token that we use for authenticat
8)multer
	This is a node.js middleware for handling multipart/form-data.
9)mysql
  for connect mysql database
10)mysql2
	alternatively using mysql database
11)nodemon
	The nodemon Module is a module that develop node. js based applications by automatically restarting the node application when file changes in the directory are detected
12)passport
13)passport-jwt
	For verify token
14)sequelize
  Makes easy to work with MySQL, MariaDB, SQLite, PostgreSQL databases, and more  
15)winston
	For a simple and universal logging library with support for multiple transports
